# REST API CRUD (FastAPI)

## Опис роботи

У даній лабораторній роботі реалізовано REST API з використанням мови Python та фреймворку FastAPI. API підтримує основні CRUD операції та роботу з кількома взаємопов’язаними сутностями.

## Мета роботи

* Ознайомитися з принципами побудови REST API
* Реалізувати CRUD операції (Create, Read, Update, Delete)
* Використати HTTP методи GET, POST, PUT, DELETE
* Реалізувати вкладені сутності
* Додати пагінацію

## Реалізовані можливості

### CRUD операції

* GET — отримання даних
* POST — створення нових записів
* PUT — оновлення даних
* DELETE — видалення записів

### Сутності

* User
* Post
* Comment

### Вкладені сутності

* User містить Posts
* Post містить Comments

### Пагінація

Реалізовано через параметри:
?page=1&page_size=5

## Інструкція запуску

1. Встановити залежності:
   pip install -r requirements.txt

2. Запустити сервер:
   uvicorn main:app --reload

3. Відкрити документацію:
   http://127.0.0.1:8000/docs
